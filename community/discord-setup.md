# AgentCost Discord Server Setup

## Server Name
AgentCost Community

## Channels

### Information
- #welcome — Auto-message with getting started links
- #announcements — Release notes, updates (announcements channel)
- #rules — Code of conduct + community guidelines

### Community
- #general — General discussion
- #introductions — New member introductions
- #show-and-tell — Share your AgentCost setups and wins

### Support
- #help — General help and troubleshooting
- #bug-reports — Bug discussion (point to GitHub Issues for tracking)
- #feature-requests — Feature discussion (point to GitHub Issues)

### Development
- #plugins — Plugin development and sharing
- #contributing — Contribution discussion
- #architecture — Technical design discussions

### Enterprise
- #enterprise — Enterprise feature discussions
- #self-hosting — Deployment and infrastructure help

## Roles
- @Maintainer — Core team
- @Contributor — Active contributors
- @Plugin Author — Published plugin authors
- @Enterprise — Enterprise users

## Bots
- GitHub bot — Webhook for releases, issues, PRs
- Welcome bot — Auto-greeting new members
