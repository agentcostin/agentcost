# Contributing

Thank you for your interest in contributing to AgentCost!

## Quick Start

```bash
git clone https://github.com/agentcostin/agentcost.git
cd agentcost
pip install -e ".[dev,server]"
pre-commit install
pytest tests/ -v
```

## Full Guide

See [CONTRIBUTING.md](https://github.com/agentcostin/agentcost/blob/main/CONTRIBUTING.md) for detailed instructions including:

- Development setup
- Project structure
- Making changes (branching, testing, linting)
- Commit convention (conventional commits)
- Pull request checklist
- Plugin development
- Enterprise feature contributions

## Ways to Contribute

- **Report bugs** — [File an issue](https://github.com/agentcostin/agentcost/issues/new?template=bug_report.yml)
- **Request features** — [Feature request](https://github.com/agentcostin/agentcost/issues/new?template=feature_request.yml)
- **Propose plugins** — [Plugin request](https://github.com/agentcostin/agentcost/issues/new?template=plugin_request.yml)
- **Improve docs** — Edit any page (click the pencil icon)
- **Submit code** — Fix bugs, add features, improve tests
